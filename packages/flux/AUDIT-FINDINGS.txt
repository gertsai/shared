  ✅ Instance Checks: Perfect Type Narrowing
     - instanceof Date → return Date (type-safe cast)
     - instanceof Map → return Map (type-safe cast)
     - instanceof Set → return Set (type-safe cast)
     - Array.isArray() → return array (type-safe cast)

  ✅ Recursive Safety: Correct
     - WeakMap threaded through all recursive calls
     - Early registration prevents infinite recursion
     - Pattern: seen.set() BEFORE recursing, not after

Code Pattern Excellence:

        deepClone(value, seen)          // ✅ Recursion with seen map
      );
    }
    return mapCopy as T;                // ✅ Type assertion justified
  }
