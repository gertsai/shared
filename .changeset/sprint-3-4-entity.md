Initial release of `@gertsai/entity` — backend-agnostic entity base classes (Model, Entity, EntityWithMetadata) with pluggable reactivity layer. Default plain-object adapter; optional Vue adapter via `@gertsai/entity/vue` subpath (peer-dep optional). Mirrors Orchestra orchlab/core entity patterns 1:1 with Vue/xid-ts/lodash dependencies stripped per ADR-005 Decision B. Per PRD-002 FR-W4-001..003.
